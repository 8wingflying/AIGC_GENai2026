## 3d
- Meta的SAM 3（Segment Anything Model 3）
  - Meta的SAM 3（Segment Anything Model 3）是其最新發布的影像理解模型，於2025年11月推出。
  - 它不僅能「看懂」圖像，還能「理解」人類語言描述的概念，並從單張2D圖像中重建出帶有文字描述的3D模型。
  - SAM 3引入了「可提示概念分割」的能力，允許用戶使用自然語言提示（如「條紋紅色雨傘」）來識別影像中的物件，並同時進行偵測、分割和追蹤。
  - 這項技術突破了前一代模型的限制，使其能夠更靈活地處理複雜的視覺任務，並為AI視覺能力的發展奠定了關鍵基礎
